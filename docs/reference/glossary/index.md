# Glossary

| Term | Definition |
|------|------------|
| **Alias** | Shortcut for a command (e.g., `alias ll='ls -lh'`) |
| **Autocomplete** | Tab completion that suggests commands/arguments |
| **Bash** | Bourne Again Shell, default Linux shell |
| **Bootstrap** | Script that sets up a system from scratch |
| **CLI** | Command-Line Interface |
| **Completion** | System that provides suggestions on Tab press |
| **Cron** | Time-based job scheduler |
| **Daemon** | Background process |
| **Dotfiles** | Configuration files (start with `.`) |
| **Fish** | Friendly Interactive Shell, modern shell |
| **fzf** | Fuzzy finder, interactive file/command search |
| **Globbing** | Pattern matching for filenames (e.g., `*.txt`) |
| **Hook** | Script that runs on Git events |
| **LSP** | Language Server Protocol, code intelligence |
| **Nerd Font** | Font patched with developer icons |
| **Oh My Zsh** | Zsh framework for managing themes/plugins |
| **P10k** | Powerlevel10k, fast Zsh theme |
| **Plugin** | Extension that adds features to a shell |
| **POSIX** | Standard for Unix-like operating systems |
| **Prompt** | Text shown before command input (e.g., `$` ) |
| **Proot** | Run Linux distros without root (user-space chroot) |
| **Starship** | Cross-shell prompt written in Rust |
| **Tasker** | Android automation app |
| **Termux** | Terminal emulator + Linux environment for Android |
| **Tmux** | Terminal multiplexer (multiple sessions) |
| **TPM** | Tmux Plugin Manager |
| **TTS** | Text-to-Speech |
| **Zsh** | Z Shell, advanced shell with many features |
