# Showcase

## Real Termux Setups

### Developer Workstation
```
Shell: Zsh + Powerlevel10k (Tokyo Night)
Font: JetBrains Mono Nerd Font
Editor: Neovim + LSP + Telescope
TMux: Tmux + TPM + Resurrect
Git: Custom aliases + hooks
Automation: Tasker + Termux:API
```

### Minimal Power User
```
Shell: Fish (built-in completions)
Font: Hack Nerd Font
Editor: Vim
Git: Oh My Zsh git plugin
Automation: Cron jobs
```

### Hacker/CTF
```
Shell: Bash with custom PS1
Font: Monospace (no Nerd Fonts)
Tools: nmap, sqlmap, metasploit
SSH: Keys + config for multiple hosts
```
