## 🚀 Quick Start

<div class="code-block" data-animate="slide-up">

```bash
# 1. Install Termux from F-Droid (recommended)
#    https://f-droid.org/packages/com.termux

# 2. Grant storage access
termux-setup-storage

# 3. Update packages
pkg update && pkg upgrade -y

# 4. Run the master bootstrap
pkg install -y curl git
curl -fsSL https://raw.githubusercontent.com/gamesiteonline/termux-mastery/main/assets/scripts/bootstrap.sh | bash
```
